# プライバシーポリシー

SeatXray（以下「本アプリ」）は、ユーザーのプライバシーを最優先し、以下の通り情報を管理します。

### 1. データの管理
個人情報（氏名、旅程、認証情報など）を収集することはありません。情報を管理するサーバーを介さず、すべてのデータはお使いのデバイスや指定した保存先にのみ記録されます。

### 2. 同期機能
データの同期を利用する場合、以下の通り保護されます。
- **保存先**: 開発者のサーバーではなく、ユーザー自身のクラウド（Google Drive, Cloudflare等）を利用します。開発者がデータにアクセスすることはできません。
- **暗号化**: データはデバイス内で暗号化してから送信されます。パスワードがない限り、外部から内容を見ることはできません。

### 3. API認証情報の取り扱い
入力された **API Key** および **API Secret** は、航空会社システムとの認証にのみ使用されます。これらの情報はデバイス内、または自身で設定した同期先にのみ保存され、第三者に共有されることはありません。

### 4. 利用規約の遵守
本アプリは Amadeus Developers の規約を遵守します。
- データの保存は円滑な表示のために一時的なキャッシュとして行いますが、規約に反する再配布や恒久的な保存は行いません。
- APIリクエストは、検索やスキャンなどのユーザー操作に応じてのみ実行され、自動では行われません。

### 5. 統計および免責
- API呼び出し回数の統計はアプリ内で完結し、外部送信は行われません。
- API通信時のデータ取り扱いは [Amadeus社のプライバシーポリシー](https://developers.amadeus.com/privacy-policy) が適用されます。
