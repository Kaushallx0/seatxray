# プライバシーポリシー

SeatXray（以下「本アプリ」）は、以下の通り情報を管理します。

### 1. データの管理

氏名や旅程、認証情報などの個人情報を収集されることはなく、情報を管理するサーバーを介さずすべてのデータはお使いのデバイスや指定した保存先にのみ記録されます。

### 2. 同期機能（将来実装予定）

クラウド同期機能は現在開発中のため、すべてのデータはローカルデバイス内にのみ保存されます。

将来的に同期機能が実装された場合でも、開発者のサーバーではなくユーザー自身のクラウド（Google Drive, Cloudflare等）を利用する仕組みとなり、開発者がデータにアクセスすることは一切ありません。
送信されるデータはデバイス内で暗号化され、第三者が内容を閲覧することは不可能な設計となります。

### 3. API認証情報の取り扱い

入力されたAPI KeyおよびAPI Secretは航空会社システムとの認証にのみ使用され、デバイス内または自身で設定した同期先にのみ保存されます。第三者に共有されることはありません。

### 4. 利用規約の遵守

Amadeus Developersの規約を遵守し、データの保存は円滑な表示のために一時的なキャッシュとして行い、規約に反する再配布や恒久的な保存は行っておりません。

APIリクエストは検索やスキャンなどのユーザー操作に応じてのみ実行され、ユーザーの予期しないタイミングで自動的に通信が行われることはありません。

### 5. 統計および免責

API呼び出し回数の統計などのデータはアプリ内で完結し、外部送信は行われませんが、通信時のデータ取り扱いは[Amadeus社のプライバシーポリシー](https://developers.amadeus.com/legal/privacy-policy)が適用されます。
