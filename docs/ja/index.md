# SeatXray

**SeatXray** は航空便の座席表を解析し、通常はグレーアウトして見ることのできない「ブロック中」や「指定済み」の座席を詳細に判別するツールです。

> 本アプリは現在開発中であり、解析ロジックが不完全な箇所や不具合が含まれている可能性があります。表示される情報の正確性については一切保証いたしません。特に、Android版は現在実験段階でありパフォーマンスや表示崩れなどが発生する場合がございます。

## 特長

- 通常は判別できない座席の詳細なステータスを可視化し、他の乗客が指定済みなのか航空会社がブロックしているのかを表示
- 外部サーバーを経由せず全データをデバイス内、またはご自身で指定したクラウドストレージにのみ保存
- WindowsとAndroidに対応し、自宅のPCでも移動中のスマートフォンでも利用可能
- 複雑な航空データを直感的な色分けで整理し、専門知識がなくても一目で空席状況を把握できるインターフェース
