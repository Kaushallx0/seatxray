# SeatXray

**SeatXray** は、航空便の座席表を解析し、通常はグレーアウトして見ることのできない「ブロック中」や「指定済み」を区別するツールです。

> 本アプリは現在開発中であり、解析ロジックが不完全な箇所や不具合が含まれている可能性があります。表示される情報の正確性については一切保証いたしません。

## 主な特長

- **座席状況の詳細解析**: 単なる選択不可ではなく、それが他の乗客によって「指定済み」なのか、航空会社によって「ブロック中」なのかを判別します。
- **プライバシー**: 外部（開発元など）にデータが送信・収集されることはありません。すべての情報は、ご自身のデバイスや指定した保存先にのみ安全に記録されます。
- **直感的な操作**: 複雑なデータを整理し、誰でも簡単に座席の空き状況を把握できます。

## 始め方

[セットアップガイド](setup.md) を参照して、APIキーの取得と設定を行ってください。

## 使用技術
- **Flet**: 高速かつモダンな画面表示を実現するフレームワーク
- **Amadeus API**: 世界中の航空券データを管理する業界標準システム
